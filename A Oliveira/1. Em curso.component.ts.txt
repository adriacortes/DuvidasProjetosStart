1. Em curso.component.ts

Inicializa o vetor assim:
//Vetor de Cursos
vetor: Curso[] = [];

2. No método seleção em curso.component.ts você não chamou o método de 'Obter Curso'!

Seu código:
 
  selecao(): void{
    this.curso_service().subscribe( // this.curso_service(). Cade a chamado do método!
      (res: Curso[]) => {
        this.vetor = res;
      } 
    )
  }


Código correto:

 selecao(){
    console.log('call selecao component');
    this.curso_service.obterCursos().subscribe(
      (res: Curso[]) => {
        this.vetor = res;
      }
    )
  }
